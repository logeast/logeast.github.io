这里记录了 Git 常用的一些命令，让通过 Git 的版本控制变得简单。

Git 常用命令

  1. 配置
    配置用户名

    $ git config --global user.name "<name>"

    配置邮箱

    $ git config --global user.email "<email address>"

  2. 创建仓库(create)
    克隆一个存在的仓库

    $ git clone <url>

    创建一个新的本地仓库，并设置名字

    $ git init <project name>

  3. 本地更改(local caches)
    跟踪文件状态，列出所有新建或更改的文件

    $ git status

    跟踪文件更改，展示没有暂存文件的差异

    $ git diff

    添加所有新增和改变的文件到版本控制

    $ git add .

    添加指定文件到版本控制

    $ git add -p <file name>

    提交文件到暂存区

    $ git commit -am <"descriptive message">

    修改最后一次提交(!不要修改已经发布的提交)

    $ git commit --amend

  4. 提交历史(commit history)
    列出当前分支的版本历史

    $ git log

    列出指定文件的版本历史

    $ git log --follow <file>

    列出谁在什么时候改变了什么文件

    $ git blame <file>

  5. 分支和标签(branches & tags)
    列出现有的分支

    $ git branch -av

    切换分支

    $ git checkout <branch>

    创建新分支

    $ git checkout -b <new-branch>

    删除新分支

    $ git checkout -d <branch>

    给本次提交打标签

    $ git tag <tag-name>

  6. 更新和发布(update & publish)
    列出所有远程主机

    $ git remote

    列出远程主机的网址

    $ git remote -v

    查看该主机的详细信息

    $ git remote show <remote>

    添加远程主机

    $ git remote add <remote> <url>

    将某个远程主机的更新全部取回本地

    $ git fetch <remote>

    取回某个远程主机的特定分支到本地

    $ git fetch <remote> <branch>

    取回远程主机某个分支的更新，并与本地指定分支合并

    $ git pull <remote> <remote-branch>:<local-branch>

    取回远程主机某个分支的更新，并与本地当前分支合并

    $ git pull <remote> <branch>

    提交本地修改到远程

    $ git push <remote> <branch>

  7. 合并和变基(merge & rebase)
    合并某个分支到当前分支

    $ git merge <branch>

    变基(!不要变基已经发布的提交)

    $ git rebase <branch>

  8. 撤销(undo)
    放弃更改并回到某个特定的提交

    $ git reset --head HEAD

    撤销所有 之后的提交

    $ git reset <commit>

最佳实践– Git 版本控制

  1. 提交关联修改(commit related changes)
  2. 经常提交(commit often)
  3. 不要提交写了一半的工作(don’t commit half-done work)
  4. 提交之前检查代码(test code before you commit)
  5. 书写好的提交信息(write good commit messages)
  6. 版本控制不是备份系统(version control is not a backuo system)
  7. 采用分支(use branch)
  8. 商定一个工作流(agree on a workflow)
  9. 帮助和文档(help & documentation)
